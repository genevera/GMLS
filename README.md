# GMLS - Google Maps Location Sharing

![](https://pp.userapi.com/c850528/v850528392/15d9f1/1FanQ51lvng.jpg)

## на Русском
### Получение местоположения друзей, которым они поделились в Google Map
Для использования нужно указать свои Cookie из Google.

Нужны только **SID, HSID, SSID** и иногда, если Google узнает, что это "скрипт", **GOOGLE_ABUSE_EXEMPTION=ID**

Для примера приведен файл ***[intable.js](https://github.com/xTCry/GMLS/blob/master/example/intable.js)*** с выводом координат в таблицу. 


## in English
### Getting the location of friends they shared on Google Map
To use, you must specify your cookie from Google.

Only **SID, HSID, SSID** are needed, and sometimes, if Google finds out that this is a "script", **GOOGLE_ABUSE_EXEMPTION=ID**
